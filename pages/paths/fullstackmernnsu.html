Full Stack Web Development MERN

Course Overview


Welcome to your web development journey! This initial stages of this course sets the foundation for everything you'll build ahead, from learning how the internet works to writing your first lines of HTML. Through practical examples, beginner-friendly content, and daily progress, you'll gain clarity and confidence in creating web pages from scratch.

What is Web Development?

Introduction to world of websites, the internet, and frontend, backend and full stack.

  • Understand how website works.
  • Difference between frontend and backend.
  • Client-server communication and browsers
Web Hosting & IDE Setup

Setting up your development environment and understanding web hosting.

  • Choose a code editor (VS Code, Sublime Text, etc.)
  • Install necessary extensions and plugins
  • Understand web hosting basics and options
Version Control with Git & GitHub and HTML Introduction

Learn the basics of version control and start writing HTML.

  • Understand Git and its importance in web development
  • Learn basic Git commands (init, add, commit, push, pull)
  • Create a GitHub account and repository
  • Introduction to HTML structure and elements
  • Creating your first HTML page
HTML Fundamentals

Dive deeper into HTML and learn essential tags and attributes.

  • se headings, paragraphs, and line breaks
  • Add images, links, and lists
  • Understand the role of attributes like href, src, alt
HTML Intermediate Level

Enhance your pages with tables, forms, and input fields.

  • Create tables for data representation
  • Build forms with various input types (text, email, password, etc.)
  • Understand form attributes and validation
HTML Advanced Level

Explore semantic HTML and advanced structure elements used in real-world websites.

  • Use semantic tags: header, nav, section, article, footer
  • Add audio and video elements
  • Understand HTML entities and iframes
