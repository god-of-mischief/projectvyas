Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It's one of the core technologies of the World Wide Web, alongside HTML and CSS.

// Your first JavaScript program
console.log("Hello, World!");

// Variables
let name = "John";
const age = 25;

// Functions
function greet(person) {
  return `Hello, ${person}!`;
}

Why Learn JavaScript?

  • It's the only programming language native to web browsers
  • Can be used for both frontend and backend development
  • Large ecosystem with countless libraries and frameworks
  • High demand in the job market

Setting Up Your Environment

To start coding in JavaScript, you need:

  1. A text editor (VS Code recommended)
  2. A web browser (Chrome, Firefox, or Edge)
  3. Node.js for running JavaScript outside the browser

Variables and Data Types

JavaScript has several ways to declare variables and multiple data types...

// Variable declarations
let mutableVariable = "I can change";
const constantVariable = "I cannot change";
var oldStyleVariable = "Avoid using var";

// Data types
let string = "Hello";
let number = 42;
let boolean = true;
let array = [1, 2, 3];
let object = { name: "John", age: 30 };
