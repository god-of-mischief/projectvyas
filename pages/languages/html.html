HTML

Structure of web pages: elements, attributes, forms, media, semantics, and accessibility.

Introduction

HTML provides the structure of web pages using elements (tags). Learn the document skeleton and basic tags.

<!DOCTYPE html>
<html lang="en">
  <head> <meta charset="utf-8"> <title>Title</title> </head>
  <body> Hello world! </body>
</html>

Elements

Elements define meaning and structure. Most elements wrap content with opening/closing tags; some are void (e.g., <img>).

Attributes

Attributes add metadata such as id, class, src, and alt. Example: <img src="photo.jpg" alt="Description">.

Text & Headings

Use <h1>â€“<h6> for headings and <p> for paragraphs. Headings form a logical outline.

Links & Images

Create links with <a href="..."> and images with <img src="..." alt="...">. Include descriptive alt text.

Lists & Tables

Use <ul>/<ol> for lists and <table> with <tr>, <th>, <td> for tabular data.

Forms

Build forms with <form>, <label>, and controls like <input>, <select>, <textarea>. Connect labels via for/id.

Semantic HTML

Use <header>, <nav>, <main>, <article>, <section>, and <footer> for better accessibility and SEO.

Media

Embed images, audio, and video with captions/tracks where appropriate. Provide fallbacks.

Accessibility

Ensure keyboard support, proper landmarks, labels, and color contrast. Use ARIA as a complement to semantics.
